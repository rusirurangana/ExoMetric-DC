ExoMetric-DC is a Discord bot designed to help you watch Minecraft servers in real time. It connects directly with your Minecraft server using the ExoMetric mod API. ## 🛠️ Setting Up and Using ExoMetric-DC Commands  

ExoMetric-DC uses straightforward commands inside Discord to make setup easy. Here are some common commands and what they do:

| Command                   | Description                          | Example                    |
|---------------------------|------------------------------------|----------------------------|
| `!setup server <IP:Port>` | Connect the bot to your server     | `!setup server 123.45.67.89:25565` |
| `!addplayer <Name>`       | Monitor a specific player           | `!addplayer Steve`          |
| `!removeplayer <Name>`    | Stop monitoring a player            | `!removeplayer Alex`        |
| `!alert enable`           | Turn on alert notifications        | `!alert enable`             |
| `!alert disable`          | Turn off alert notifications       | `!alert disable`            |
| `!dashboard`              | Show the dashboard link or info    | `!dashboard`                |
| `!help`                   | List all available commands         | `!help`                    |

You can enter these commands in any Discord channel where the bot has permission to read and send messages.
